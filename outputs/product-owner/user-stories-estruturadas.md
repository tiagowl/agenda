# User Stories Estruturadas

## US-01 - Login da aplicacao
**Como** usuario do sistema  
**Eu quero** fazer login com credenciais validas  
**Para que** apenas eu acesse meus compromissos.

## US-02 - Criar compromisso
**Como** usuario do sistema  
**Eu quero** cadastrar compromissos com nome, data, horario e observacoes  
**Para que** eu consiga planejar minha rotina com contexto completo.

## US-03 - Editar compromisso
**Como** usuario do sistema  
**Eu quero** editar qualquer campo de um compromisso ja criado  
**Para que** eu mantenha minha agenda sempre atualizada.

## US-04 - Excluir compromisso
**Como** usuario do sistema  
**Eu quero** remover compromissos nao mais necessarios  
**Para que** a agenda reflita apenas itens relevantes.

## US-05 - Visualizacao diaria/semanal/mensal
**Como** usuario do sistema  
**Eu quero** alternar entre visoes diaria, semanal e mensal  
**Para que** eu enxergue meus compromissos com diferentes niveis de detalhe.

## US-06 - Persistencia local
**Como** usuario do sistema  
**Eu quero** que meus dados fiquem salvos no dispositivo  
**Para que** eu nao perca compromissos ao fechar ou recarregar a aplicacao.

## US-07 - Exportar dados em JSON
**Como** usuario do sistema  
**Eu quero** exportar meus dados para arquivo JSON  
**Para que** eu tenha backup da minha agenda.

## US-08 - Importar dados em JSON
**Como** usuario do sistema  
**Eu quero** importar um arquivo JSON valido de agenda  
**Para que** eu recupere ou migre meus dados rapidamente.

## US-09 - Dashboard de acompanhamento diario
**Como** usuario do sistema  
**Eu quero** ver no dashboard os compromissos do dia, total marcado e total nao realizado  
**Para que** eu acompanhe minha execucao diaria com clareza.
